# Equipe 2-03 -  SAE S301 - Développement d'une application web dynamique

<img src="./sujet/Logo_IUT.jpeg" width="300"  />

Ce projet est une SAÉ : Situation d'Apprentissage et d'Évaluation, dans le cadre du Semestre 3 du BUT Informatique de l'IUT de Nantes.

Projet: Nous devons concevoir et développer une application web dynamique de présentation et de vente des produits d'une entreprise.
Nous utiliserons pour cela le framework PHP CodeIgniter, ainsi qu'un SGBD relationnel (sqlite).
